# Pacman Game

## How to Play

1. [Create a new issue](https://github.com/Abhinand-s/Abhinand-s/issues/new?title=Move+Pacman+to+(X,Y)).
2. In the issue title, replace `X` and `Y` with the coordinates where you want Pacman to move.
3. Submit the issue.
